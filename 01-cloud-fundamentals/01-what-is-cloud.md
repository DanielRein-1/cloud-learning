# Basics of Cloud Computing

## What is Cloud?

The cloud refers to a network of remote servers hosted on the internet that store, manage, and process data instead of using a local computer or personal server.

In practice, the cloud allows users to access computing resources such as storage, applications, and processing power from anywhere, as long as there is an internet connection.

These resources are maintained in large-scale data centers operated by cloud providers such as Google Cloud, Amazon Web Services, and Microsoft Azure.

---

## What is Cloud Computing?

Cloud computing is the on-demand delivery of computing services over the internet, including servers, storage, databases, networking, software, and analytics.

Instead of owning physical infrastructure, users rent resources from cloud providers and pay based on usage.

This model enables:

- faster deployment of applications
- reduced infrastructure costs
- scalable and flexible computing resources

Cloud computing is typically delivered through three main service models:

- Infrastructure as a Service (IaaS)
- Platform as a Service (PaaS)
- Software as a Service (SaaS)

---

## Deployment Models

### Public Cloud

A public cloud is a computing environment where services are delivered over the public internet and shared across multiple customers.

- Infrastructure is owned and managed by a cloud provider
- Resources are shared (multi-tenant environment)

**Example:**
Google Cloud Platform (GCP), Amazon Web Services (AWS), Microsoft Azure

---

### Private Cloud

A private cloud is a computing environment dedicated to a single organization.

- Infrastructure is not shared with other organizations
- Offers greater control and security

**Example:**
A company hosting its internal systems in a dedicated data center or private cloud infrastructure

---

### Hybrid Cloud

A hybrid cloud combines both public and private cloud environments.

- Sensitive workloads may run in private cloud
- Scalable workloads run in public cloud

This approach allows organizations to balance security, cost, and flexibility.

---

## In a Nutshell

- Cloud = Remote computing resources delivered over the internet
- Cloud computing = Renting computing power instead of owning hardware
- Public cloud = shared infrastructure
- Private cloud = dedicated infrastructure
- Hybrid cloud = combination of both
